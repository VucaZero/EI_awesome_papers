# awesome_papers

本目录是所有协作者上传论文笔记、主题总结与阶段性分析的主存放位置。

## 使用原则

- 按主题缩写进入对应文件夹
- 单篇精读放 `paper_notes/`
- 主题总结放 `topic_summaries/`
- 新主题必须先在根目录 `TAXONOMY.md` 中补充定义
